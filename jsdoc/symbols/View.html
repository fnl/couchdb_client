query(parameters, keys, callback)
Query the view. Note that the requests that serialize to JSON always first buffer the response data, until the reponse ends and then return the JSON object. If you wish to make a streaming query of a view, you will need to use the CouchClient#request method and decide yourself when to parse parts of the chunks to JS objects.
